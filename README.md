# Employee Payroll Tracker

## Overview

The Employee Payroll Tracker is a web application designed to help payroll managers view and manage employee payroll data efficiently. This application allows managers to add employee details such as first name, last name, and salary, sort the employees alphabetically by last name, and view aggregated payroll data through the console.

## Description

This application was built to provide a user-friendly interface for payroll managers to keep track of their employees' payroll data. The app runs in the browser and dynamically updates the HTML and CSS using JavaScript. It features a clean, responsive design that adapts to multiple screen sizes, ensuring usability on various devices.

## Changes Made

Implemented the core functionality to add employees to a payroll tracker.
Added JavaScript functionality to prompt users for employee details (first name, last name, and salary) and dynamically update the employee roster displayed on the page.
Developed a sorting feature that arranges the employee list alphabetically by last name.
Included a console log to display computed and aggregated payroll data.
